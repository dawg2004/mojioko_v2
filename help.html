// PLAUD TOKEN 取得ガイド

Googleアカウントでログインしている方向け

▶ 方法① DevTools コンソール（PC推奨）

Chromeの開発者ツールを使ってトークンを取得します。手順通りにやれば3分で完了。

STEP 1
web.plaud.ai → を開いてGoogleでログイン
すでにログイン済みならそのままでOK
▼
STEP 2
DevToolsを開く
方法A：ページ上で右クリック →「検証」をクリック
方法B：キーボードショートカット F12（WindowsはF12、Macは⌥⌘I）
▼
STEP 3
Console タブをクリック
画面上部のタブから「Console」を選択
▼
STEP 4
ペースト許可を有効にする（初回のみ）
コンソールの > の横に以下を手打ちして Enter
DevTools — Console
⚠ Warning: Don't paste code into the DevTools Console...
　Please type 'allow pasting' below and press Enter.
> allow pasting ← ここに手打ち → Enter
⚠️ 「allow pasting」はコピペ不可、キーボードで手打ちしてください。
Chromeのセキュリティ仕様です。一度やれば以降は不要です。
▼
STEP 5
下のコードをコピーして、コンソールに貼り付けて Enter
COPYボタンを押してからコンソールに ⌘V / Ctrl+V で貼り付け
JSON.parse(localStorage[Object.keys(localStorage).find(k=>k.includes('workspaceList'))]||'[]')[0]?.workspaceToken
▼
STEP 6
トークンが表示される → コピー
'eyJhbGci...' のような長い文字列が表示されます
DevTools — Console
> JSON.parse(localStorage[Object.keys(localStorage)...
< 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJhbGci...'
　 ↑ この文字列を全部コピー（クリックして選択 → ⌘C）
▼
STEP 7
mojioko に戻って TOKEN タブに貼り付け → ▶ LINK
録音リストが表示されれば接続完了！
✅ 接続完了！録音リストが mojioko に表示されます。
トークンは自動保存されるので、次回以降は再ログイン不要です。
OR

▶ 方法② PLAUDにパスワード設定（スマホ対応・永久解決）

一度パスワードを設定すれば、スマホ・PCどこからでもメールでログインできます。

STEP 1
PLAUDの設定ページ → を開く
web.plaud.ai にログイン済みの状態で開く
▼
STEP 2
「アカウント」→「パスワード設定」から新しいパスワードを設定
登録メールアドレス宛に確認メールが届く場合があります
▼
STEP 3
mojioko の PLAUD LOGIN タブでメール＋パスワードでログイン
スマホでも毎回ワンタップでOK！
✅ これが一番スムーズ！一度設定すればスマホからでもログインできます。
